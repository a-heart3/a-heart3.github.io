关于学习

发表于 2024-11-05 更新于 2024-12-01 分类于 想法
本文字数： 2.4k 阅读时长 ≈ 9 分钟

本篇blog用于介绍自己关于学习的一些想法。

首先，什么是学习？

在我看来，学是模仿，习是习以为常。即我们要从他人，从自然界看到一种现象，并且不断模仿，最后让自己习惯这种现象。

当然，上述只是我粗浅的理解，更深入一些的理解则为:模仿—>熟练掌握—>创新输出。这三个步骤缺一不可。新奇的想法，伟大的想法，提出的前提是对前人知识的熟练掌握，再在此基础上提出自己的创新。亦或是对自然有浓厚的好奇心，在不断的观察和模仿中掌握某些知识，并提出新的观点。

那么，仅仅是传统的理工科，文科的学习才能是学习吗？其实并不是的，舞蹈，体育，美术等艺术的表现形式也是学习。在工作社会中如何与人交往，如何与他人进行有效的沟通，这也是学习。街边的小吃怎么做才能更好吃等等这些都是学习。

学习的内容广，光在一门学问上有所深造已经是极难的事情了，如果有人对很多内容都有很深的研究，那他就十分值得我们敬佩。

这些就是我自己对学习的理解。

那么，为什么要学习？

学习的目的可以有伟大的，比如“为中华之崛起而读书”，也可以有平凡的，即“找到一份好的工作，以后有一个好的生活”。对于以前一种目的学习的人，我们需要敬仰。因为祖国的发展，科学技术的进步需要仰丈他们。对于以后一种目的学习的人，我们也应该尊重，向往美好的生活本就是人类乃至生物的天性，安安稳稳，积极向上的生活也是在为国家做贡献。

其实，在如今的社会，学习对于我们来说，更需要满足第二种目的。ai的快速发展，在可预见的未来，可能会淘汰掉越来越多的劳动力。所以，要想不被淘汰掉，我们需要不断的学习，不断的精进自己，才能为自己博一个好的未来。

其次，学习带来的也不仅仅是金钱，地位的提升，以及一种本能的满足感。人类从生下来就有求知欲，不然，为什么大家都觉得小孩子脑子里的问题多？稀奇古怪的想法多呢？在学生阶段，总会有那么几次，你会为自己千辛万苦解出来的问题而欢呼雀跃，总会想疯狂得向别人展示你千辛万苦学来的体育运动技术。这就是学习带给我们的满足感，相较于一些没有营养的短视频，烂片，烂游戏（注意：我这里说的是烂片，烂游戏等等，好的电视剧，游戏总会给你一些思考，让你去想了解一些东西），这种满足感是持久的。

再就是思维方式的改变，这点在我来看就是独立思考的能力，以及求解真相的能力。比如，现在由于自媒体的快速发展，以及短视频的推荐算法，会导致很多人容易变得人云亦云，被虚假的新闻报道带偏了自己的判断，并被困在了信息茧房之中。这无疑是一种悲哀，每个人都是一个独立的个体，既然独立，那么就应该有自己的思考和判断，而不是轻易的被带走了自己的想法。这种时候，自己学习到的知识就很有说服力。有独立思考的能力是可贵的，虽然在日常生活中偶尔被带偏思考无所谓，但是在自己的所学领域内被带歪是一件可怕的事，可能会给你带来损失。（这就好比有人装逼装到了自己擅长的领域，总规是能看个乐子嘛）

最后，其实学习也会带来想法的改变，但这也是一些人的痛苦的来源。自己辛辛苦苦，寒窗苦读很多年，发现不如投胎投的好的同学。在市场的大环境下，工资和生活水平可能不如学历不如自己的一些人。优秀的人个方面都很优秀带来的压力。这些确实无法避免，人生的发展与机遇也有很大的关系。我也曾被这些困惑，也曾质疑过学习的作用，后来我想了想，分析了自己的条件和性格，发现自己如果不学习，过得并不会比现在好，甚至可能会差很多。于是，我放平了心态，我只需要有自己喜欢的事，有自己喜欢的人，每天过得再好一点点就行了。这也是学习给我带来的，相较于不上学，我能够看到更多不同生活状况的同学，与他们交朋友，听他们的看法，对自己的想法也在不断改变，最后慢慢形成自己的生活观。

最后，怎么学习？

首先，学习的范围很广，如果都只懂皮毛的话，其实也没有什么意义。最好的选择就是选两三门自己特别感兴趣的学科认真学习，学的深入一些。在常年的文理科纷争当中，我一直支持两者都学（即理科也要了解一些文科知识，文科也要了解一些理科知识）,知识，艺术这些都是相通的，都有他们的美感。

关于学习方向，我现在能想到也就这些。关于学习的方法，我想大概分为几个方面。

  1. 老师讲解：老师讲解我觉得是入门非常重要的一环，好的老师可以激起学生对于该学科的兴趣，并能够将学科脉络，重点思想理清，使得学生一开始并不需要过多的深入细节而迷失。（大佬可能直接进行下一步了）
  2. 读书，做实践：书中会对知识的细节有所补充，实践（对于工科来说，可能是课程lab），是帮助自己更加深入理解知识，培养自己发现问题，解决问题方法的利器。
  3. 做输出:即写报告，将老师所讲，读书所得，实践所得总结，理清，加入自己的理解，写成新的报告，文章，博文分享（输出）
  4. 创新:在做实验时遇到的问题，读书上课想到的问题，看看自己能不能做解决优化。以批判性思维学习，不仅要看别人讲了什么，还要看看别人在文章中没讲什么（这些可能就是创新的点）。

关于学习读书的步骤，我个人是分成了以下几个部分:

  1. 听课，老师讲的，提到的大多是重点，需要在书中重点看。
  2. 读书（书读三遍）
    1. 首先是通篇概览，重点关注老师课上讲的部分，将自己的思考，疑惑批注下来，方便以后提问或查找解答。
    2. 结合实践不停翻阅：实践中遇到的问题要记录，实践中经常出现的问题也是重点，如何解决实践中的问题，解决思路要及时记录下来。
    3. 输出时翻阅：此时应将旁注中的思考结合实践整理，将疑惑通过问老师和同学，或者上网查资料解决，整理输出。

关于解决创新性问题，我目前能想到的方法是以下几点（完全借鉴《怎样解题》）

  1. 我们可以把创新点看作一道题目，其实就是怎么解题
  2. 首先，要充分理解题目，即我需要知道
    1. 我的目标是什么。
    2. 我现在有哪些知识和条件可以用到
    3. 我现在有哪些数据可以用到。
  3. 找到已知数据，条件，知识和目标的联系
    1. 类比：以前有没有解决过类似的题目，是怎么解决的？这次跟之前的问题的条件和数据有什么不同？
    2. 辅助：辅助元素，辅助题目，能不能帮助我们更好的理解题目？
    3. 大问题分解成不同的小问题来解决（能分解为相似的小问题最好了）
    4. 条件的排列组合，看能不能做出一些新的花样来
    5. 问题导向：解决这个问题，我需要什么数据和条件，那么这个数据和条件我有吗？如果没有的话，我可以通过哪些数据和条件获得呢？一步步逆向推导
  4. 制定解决方案并执行：
    1. 分模块，分步骤
    2. 重点检查主要步骤，然后去看细节无误
  5. 检验：
    1. 正向检验，每一个数据都应该能够得到你要的条件和数据，直到结果
    2. 想想还有没有其他办法？
    3. 总结记录。

以上就是目前我关于学习的想法，也是我自己希望自己能达到的学习状态。
